Jeu en Console - Castle Of Hyrule

Technologie utilisée : Langage de programmation utilisé TypeScript avec NodeJS

Répertoire Git : Lien vers le référentiel Git

Description : Ce projet avait pour objectif de créer un jeu en console où le joueur explore un château sur plusieurs étages. Le personnage du joueur et le monstre qu'il affronte sont choisis aléatoirement à partir d'un fichier JSON. Un boss est également sélectionné de manière aléatoire tous les 10 étages.
