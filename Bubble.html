Site de vente - Bubble My Tea

Technologie utilisée : Framework utilisé Laravel et Bootstrap. Base de données MySql

Répertoire Git : Lien vers le référentiel Git

Description : Le principe de ce projet était de créer un site de vente de Bubble Tea avec deux types de comptes, un pour les utilisateurs et un autre pour les administrateurs, chacun ayant des fonctionnalités différentes. Les utilisateurs peuvent acheter des produits, consulter leur historique de commandes, accéder à la liste des produits et modifier leur profil. Les administrateurs, quant à eux, ont la possibilité d'ajouter des produits au catalogue.
